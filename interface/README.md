<h3 align="left">INTERFACE</h3>
<p align="left">

This folder contains the examples of the library usage:
- ```demo.py```: demonstration file that runs all library modules on the selected dataset to collect data on pretrained models outputs;
- ```gui.py```: demonstration file with graphical user interface to receive and save reports on submitted images.

It is required to run them from Python IDE.
</p>
