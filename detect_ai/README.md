<h3 align="left">DETECT AI</h3>
<p align="left">

This folder contains the source code of the main modules of the library:
- ```generationg```: source code for the detection of generated images containing a person's face;
- ```modifying```: source code for the detection of modified images containing a person's face;
- ```swapping```: source code for the detection of images containing a person's face with a faceswap;
- ```preprocessing```: source code for the preprocessing of images containing a person's face;
- ```reporting```: source code for the generation of reports about detection.
</p>
