Данынй каталог содержит исодный код основных модулей библиотеки:
- ```generationg```: исходный код для обнаружения сгенерированных изображений, содержащих лицо человека;
- ```modifying```: исходный код для обнаружения модифицированных изображений, содержащих лицо человека;
- ```swapping```: исходный код для обнаружения изображений с подменой лица человека;
- ```preprocessing```: исходный код для предварительной обработки изображений, содержащих лицо человека;
- ```reporting```: исходный код для генерации отчетов об обнаружении.
