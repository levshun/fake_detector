<h3 align="left">TUTORIALS</h3>
<p align="left">

This folder contains tutorials that are explaining how to work with the library:
- ```generating.en.ipynb```: how to detect generated images containing a person's face;
- ```modifying.en.ipynb```: how to detect modified images containing a person's face;
- ```swapping.en.ipynb```: how to detect images containing a person's face with a faceswap.
</p>
