<h3 align="left">TUTORIALS</h3>
<p align="left">

This folder contains tutorials that are explaining how to work with the library:
- ```generating.ipynb```: how to detect generated images containing a person's face;
- ```modifying.ipynb```: how to detect modified images containing a person's face;
- ```swapping.ipynb```: how to detect images containing a person's face with a faceswap.
</p>
